Prevents registration with specified email domains and email addresses, that you can add in the Plugin settings.

== Description ==
This WordPress plugin allows you to block user registrations from specific email addresses and domains.
By adding unwanted email addresses and domains to your blacklist, you can prevent spammers from registering on your site. 
When a user tries to register with an email or domain on the blacklist, they will receive an error message. 
The plugin also provides statistics on the number of blocked registration attempts.
